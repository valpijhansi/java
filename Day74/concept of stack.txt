Stack :
      The Java collections framework has a class named Stack that provides the functionality of the stack data Structure.
      The Stack calss extends the Vector class.
      In stack,elements are stored and accessed in "Last In First Out" manner.
      That is, elements are added to the top of the stack and removed from the top of tge stack.

Stack Operations : 
                    1.push()
                    2.pop()
                    3.peek()
                    4.search()
                    5.empty()

Applications of Stack :
                        1.Evaluation of Arithmetic Expressions
                        2.Backtracking
                        3.Delimiter Checking
                        4.Reverse a data
                        5.Processing Function Calls
